# RubyRulesBot

Telegram bot to enjoy Ruby programming language.

### Usage

Add [RubyRulesBot](https://telegram.me/RubyRulesBot) and send any sentence:

    /puts "Hello World!"  # Hello World!
    /puts [1,2,3,4].map{|x| x*2}   # [2, 4, 6, 8]
    /puts a = []; a.push 5   # [5]
